# Landing Zone AI

An AI-powered system for analyzing aerial imagery to identify safe landing zones.

## Structure
- `client`: React Frontend
- `server`: Node.js Backend
- `python-ai`: AI Inference Service
- `docs`: Documentation

## Getting Started

### Prerequisites
- Node.js
- Python 3.9+
- MongoDB

### Prerequisites & Setup
**See [PREREQUISITES.md](./PREREQUISITES.md) for a detailed guide on setting up the environment without Docker.**

### Quick Start (Manual)

1. Client
\`\`\`bash
cd client
npm install
npm run dev
\`\`\`

2. Server
\`\`\`bash
cd server
npm install
npm run dev
\`\`\`

3. Python AI
\`\`\`bash
cd python-ai
pip install -r requirements.txt
python app.py
\`\`\`
